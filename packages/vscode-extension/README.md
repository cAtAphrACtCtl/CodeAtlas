# CodeAtlas VS Code Extension

This extension provides the VS Code command surface for the local-first CodeAtlas platform.

## Commands

- `CodeAtlas: Register Current Workspace`
- `CodeAtlas: Discover Repositories`
- `CodeAtlas: Show Repository Status`
- `CodeAtlas: Open CodeAtlas Config`

## Development

From the repository root:

```bash
npm run extension:build
```

Then launch the `Run CodeAtlas Extension` debug configuration.
